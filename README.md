# 门店扫码

手机浏览器扫码工具：按 Excel 编码表验证二维码，为当前门店计数，并在本机数据库中全局拦截重复二维码。

## 直接使用（推荐）

双击打开 `门店扫码.html`，无需安装或启动服务。

## 开发模式

在 PowerShell 中运行：

```powershell
.\start.ps1
```

浏览器会打开 `http://localhost:4173`。相机功能需要浏览器允许摄像头权限，推荐最新版 Chrome。

## 数据说明

- 门店及扫码记录保存在当前浏览器的 IndexedDB 中。
- 同一二维码即使切换门店也不会再次计数。
- 页面右上角可将全部记录导出为 CSV。
- 浏览器数据被清除时本地记录也会消失；正式多人使用时应连接服务器数据库及账号系统。


## 手机浏览器扫码

实时扫码使用本地打包的 html5-qrcode 2.3.8（Apache-2.0），支持 Android Chrome/Firefox 及 iOS 15.1+ Safari/Chrome。相机必须通过 HTTPS 页面调用；若实时相机受限，可使用“拍照识别”。

- 当前门店可一键复制报告，或通过手机系统分享面板转发到 WhatsApp、微信等应用。
